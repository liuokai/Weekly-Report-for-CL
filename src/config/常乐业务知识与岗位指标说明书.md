# 常乐业务知识与岗位指标说明书

1. 业务概览

- **公司属性**：常乐是一家连锁推拿按摩服务提供商。
- **经营模式**：在全国重点商场开设直营店，招聘自有员工提供标准化服务。
- **用户路径**：用户通过微信小程序完成线上预约（含门店、时段、项目及推拿师选择），随后到店接受服务。

4. 管理监控逻辑

- **纵向穿透**：上级管理层不仅背负自身核心指标，还需监控下属及关联岗位的指标。例如：熊生兵（总监）监控全局指标；技术副总需监控“推拿师产值达标率” 。
- **风险预警应用**：当特定指标发生异常时，AI 应自动穿透至上述对应的责任人及其上级。
